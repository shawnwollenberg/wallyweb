Project Aegis commerce platform

Cortex

The a-commerce layer for autonomous agents: Project Aegis smart account guardrails, service discovery, delegated budgets, payment terms, receipts, disputes, and machine-readable protocol state.

Cortex gives agents and merchants a shared way to publish services, commit quote terms, execute payments, and record the signals that make commerce auditable. It combines Project Aegis agentic smart accounts with commerce workflows for wallet transfers, ERC-20 transfers, swaps, facilitator-mediated settlement, and x402 without requiring a new chain.

  • Aegissmart accounts
  • a-commerceagent commerce
  • API + MCPagent access
01 Discover

Merchants publish service records with metadata, capabilities, payout context, and active status.

02 Authorize

Owners configure Aegis guardrails, agent budgets, allowlists, swap permissions, facilitator budgets, and x402 policy.

03 Commit

Quotes bind service, agent, token, amount, payment rail, expiry, resource hash, and terms hash.

04 Settle

Receipts, disputes, solver activity, and analytics become indexed state for agents and dashboards.

walletLens

A paid x402 API and local MCP server for AI agents that need normalized EVM wallet portfolio data.

Agents can call walletLens directly for token balances, USD pricing, recent activity, portfolio summaries, stablecoin exposure, top positions, and data-coverage warnings for any 0x address across five EVM chains.
