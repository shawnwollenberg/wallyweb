Base-native commerce protocol

Cortex

The commerce layer for autonomous agents on Base: service discovery, delegated budgets, payment terms, receipts, disputes, and machine-readable protocol state.

Cortex gives agents and merchants a shared way to publish services, commit quote terms, execute payments, and record the signals that make commerce auditable. It supports wallet transfers, ERC-20 transfers, swaps, facilitator-mediated settlement, and x402 without requiring a new chain.

  • Basenative network
  • 0 bpsprotocol fees
  • API + MCPagent access
01 Discover

Merchants publish service records with metadata, capabilities, payout context, and active status.

02 Authorize

Owners configure agent budgets, allowlists, swap permissions, facilitator budgets, and x402 policy.

03 Commit

Quotes bind service, agent, token, amount, payment rail, expiry, resource hash, and terms hash.

04 Settle

Receipts, disputes, solver activity, and analytics become indexed state for agents and dashboards.
